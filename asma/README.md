# Projeto Asma - Comunicação Clínica

Guia de Comunicação Clínica em Asma: Ferramentas Baseadas em Evidências

## Estrutura

- `guia_comunicacao_clinica_v8.html` - Guia completo standalone (HTML)

## Ferramentas Incluídas

1. **Entrevista Motivacional** (g=0.28, 119 RCTs)
2. **Empatia Clínica** (r=0.23, 55 estudos)
3. **ICE - Ideas, Concerns, Expectations** 
4. **Ask-Tell-Ask**
5. **Teach-Back** (OR=0.55, NNT~5)
6. **Comunicação Não-Verbal**
7. **Linguagem Simples / Health Literacy**
8. **Comunicação de Risco**

## Fontes

Todas as recomendações são baseadas em evidências Tier 1:
- Meta-análises
- Revisões sistemáticas
- Guidelines (GINA, NICE, WHO)
